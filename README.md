## 🎯 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/cdmtn-dev/codemotion-ide.git

# Navigate to the project
cd codemotion-ide

# Install dependencies
npm install

# Start the development server
npm start
```
